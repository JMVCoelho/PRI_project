to run:
./python exercise_2

It'll the 5 models specified in the report (takes a long time...)

/Dataset_original ----------------------- contains xml version of the dataset
/Dataset_as_txt ------------------------- contains the original dataset (in txt, already parsed the xml)
/Dataset_preprocessed ------------------- contains the preprocessed version of the dataset
/Keyphrases_experimental_<MODEL> -------- contains the keyphrases generated by the program, for the specified <MODEL>
/Keyphrases_golden_set ------------------ contains the golden set of keyphrases for the dataset (after parsing the json)
/PR_<MODEL> ----------------------------- contains serialized objects: for each file, a dictionary with the PR(<MODEL>) scores for each candidate
.test.reader.json ----------------------- contains the original version of the keyphrases
