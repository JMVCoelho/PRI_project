to run:
./python exercise_3

It'll run the top-scoring set of features: w_tfidf + pagerank(W:OC; P:IDF)

/Dataset_as_txt ------------------------- contains the original dataset (in txt, already parsed the xml)
/Dataset_preprocessed ------------------- contains the preprocessed version of the dataset
/Keyphrases_experimental ---------------- contains the keyphrases generated by the program
/Keyphrases_golden_set ------------------ contains the golden set of keyphrases for the dataset
/Pickles -------------------------------- contains serialized objects: for each file, a dictionary with the PR(W:OC; P:IDF) scores for each candidate
